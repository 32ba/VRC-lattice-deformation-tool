## Algorithm Reference

The bone weight recalculation feature is based on:

**Robust Skin Weights Transfer via Weight Inpainting**
Rinat Abdrashitov, Kim Raichstat, Jared Monsen, David Hill
SIGGRAPH Asia 2023

The package uses a paper-inspired cotangent harmonic approximation rather than the paper's full mixed-energy formulation.

- Paper: https://www.dgp.toronto.edu/~rinat/projects/RobustSkinWeightsTransfer/
- Reference Implementation (MIT License): https://github.com/rin-23/RobustSkinWeightsTransferCode
